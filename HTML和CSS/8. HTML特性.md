## BFC规范

触发bfc规范的条件

- 绝对定位元素：position（absolute、fixed）
- 浮动元素：float除none以外的值

- display 为 inline-block、table-cells、flex

- overflow除了visible以外的值（hidden、auto、scroll）

BFC规范能解决那些问题

> - 解决margin传递问题
> - 解决margin叠加问题
> - 解决浮动问题
> - 解决覆盖问题
