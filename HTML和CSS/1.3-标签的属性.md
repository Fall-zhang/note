### input标签

> 在元素内的可选属性

| input 中 type 属性可选值 | 作用                               |
| ------------------------ | ---------------------------------- |
| radio                    | 定义圆形选择框（单选）             |
| checkbox                 | 方形选择框（复选框）               |
| button                   | 可定义按钮                         |
| file                     | 定义输入字段和上传按钮             |
| reset                    | 重置表单中的数据                   |
| submit                   | 可以把数据提交到服务器             |
| text                     | 单行的输入字段，默认宽度为20个字符 |
| hidden                   | 隐藏的输入字段                     |
